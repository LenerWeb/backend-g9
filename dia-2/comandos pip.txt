pip install NOMBRE_LIBRERIA > instalar la libreria, sino la encuentra lanzara un error
python -m pip install --upgrade pip > actualizar pip (gestor de paquetes ) a la ultima version
pip freeze                          > lista todas las librerias instalada en el scope actualizar
pip lista                           > lista todas las librerias (intaladas predeterminadamente y manualmente)
